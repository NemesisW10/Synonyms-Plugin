## Description:

This is a plugin for TinyMCE WYSIWYG rich text editor. On clicking the lookup synonyms button you will find an interactive popup, wherein you can type your words and find their synonyms and other related words which might be helpful. Upon submitting, the plugin fetches all the synonyms from Datamuse API and displays them in the popup. Select the synonym that you like and it gets inserted into the editor. Voila you just created a better article !
